## ğŸ”Œ Backend Integration

### API Endpoints

The frontend communicates with the Flask backend running on `http://localhost:5000`:

#### 1. Submit Case for Verdict

- **Endpoint**: `POST /verdict`
- **Purpose**: Get AI verdict for a case
- **Request Body**:
  ```json
  {
    "plaintiff": "Plaintiff statement...",
    "defendant": "Defendant statement...",
    "evidence": "Additional evidence..."
  }
  ```
- **Response**:
  ```json
  {
    "winner": "Plaintiff/Defendant/Neutral",
    "confidence": "high/medium/low",
    "model": "AI Judge ML Model",
    "plaintiff_score": 8,
    "defendant_score": 5
  }
  ```

#### 2. Generate Reasoning

- **Endpoint**: `POST /api/genai_reason`
- **Purpose**: Generate detailed legal reasoning
- **Request Body**:
  ```json
  {
    "plaintiff": "Plaintiff statement...",
    "defendant": "Defendant statement...",
    "evidence": "Additional evidence...",
    "verdict": "Plaintiff"
  }
  ```
- **Response**:
  ```json
  {
    "reasoning": "Detailed legal reasoning...",
    "model": "Local GenAI (Phi-3 Mini)"
  }
  ```

#### 3. Health Check

- **Endpoint**: `GET /health`
- **Purpose**: Check backend status

### API Client (`lib/api.ts`)

The API client provides TypeScript interfaces and functions for backend communication:

```typescript
import { submitCase, generateReasoning, checkBackendHealth } from "@/lib/api"

// Submit a case
const verdict = await submitCase({
  plaintiff: "...",
  defendant: "...",
  evidence: "..."
})

// Generate reasoning
const reasoning = await generateReasoning({
  plaintiff: "...",
  defendant: "...",
  evidence: "...",
  verdict: "Plaintiff"
})
```

## ğŸ“ Creating a New Case

1. Navigate to **"New Case"** from the navigation menu
2. Fill in the form:
    - **Case Title**: A descriptive title for the case
    - **Plaintiff Statement**: The plaintiff's claims and arguments
    - **Defendant Statement**: The defendant's defense and counter-arguments
    - **Additional Evidence**: (Optional) Supporting documents or information
3. Click **"Submit Case for AI Judgment"**
4. The AI will:
    - Analyze both statements
    - Generate a verdict (Plaintiff/Defendant/Neutral)
    - Create detailed legal reasoning
    - Calculate scores for both parties
5. View the complete verdict on the case detail page

## ğŸ› Troubleshooting

### Backend Connection Issues

If you see "Cannot connect to backend server":

1. Ensure the Flask backend is running:
   ```bash
   cd "Code Vibers\Backend"
   python app.py
   ```
2. Check that the backend is on port 5000
3. Verify firewall settings
4. Check browser console for CORS errors

### Installation Errors

If `npm install` fails:

```bash
# Use legacy peer deps flag
npm install --legacy-peer-deps
```

This resolves React 19 compatibility issues with some packages.

### Port Already in Use

If port 5173 is in use:

1. Change the port in `package.json`:
   ```json
   "dev": "next dev -p 3000"
   ```
2. Or kill the process using the port
